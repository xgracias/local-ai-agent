# local-ai-agent

Local AI Agent with RAG

## step 1 - setup env

windows:

```bash
python -m venv venv
./venv/Scripts/activate
pip install -r .\requirements.txt
```

## step 2 - install ollama

https://ollama.com/

## step 3 - pull models

```bash
ollama pull llama3.2
ollama pull mxbai-embed-large
```

## step 4 - run the app

```bash
python ./main.py
```
